Contents:
- data/: data.zip contains: 

	- anes_timeseries_2020_stata_20220210: 2020 ANES Survey retireved from: https://electionstudies.org/data-center/2020-time-series-study/

	- anes_timeseries_2024_stata_20250430: 2024 ANES Survey retrieved from: https://electionstudies.org/data-center/2024-time-series-study/

	- 2020_prepped: ANES_Project_01_Prep
      2024_prepped: Generated by ANES_Project_01_Prep

    - final_prepped: merge of 2020_prepped and 2024_prepped; used for regression analysis


- code/: Four do-files
    - ANES_Project_00_Execute.do
	- ANES_Project_01_Prep.do
	- ANES_Project_02_Summarize.do
	- ANES_Project_03_Estimate.do
    

- results/: 
	- Raw summary statistics table and regression tables as .tex files

-figures/:
    - Distrubtions of sample social media usage and political interest as .png files

- logs/: Stata output logs

Replication Instructions: 

1. Clone this repository
2  Unzip contents of data.zip to data folder
3. Update cd in each do file to reflect local repo location
4. Run do files in this order:

    1. ANES_Project_01_Prep.do
	2. ANES_Project_02_Summarize.do
	3. ANES_Project_03_Estimate.do

    OR

    1. ANES_Project_00_Execute.do (runs the above)
